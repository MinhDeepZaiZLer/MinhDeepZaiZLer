## ⭐ Highlight Projects

### 📝 Smart Study Desk (IoT & Productivity)
An embedded system built on TivaC Launchpad that serves as an intelligent workstation assistant.
- **Features:** Real-time environment monitoring (Temp/Humid/Light), Embedded Pomodoro Timer FSM, and a custom **Focus Score** algorithm.
- **Connectivity:** Streams multi-sensor telemetry data over MQTT via WiFi AT Commands interface directly to **ThingsBoard Dashboard**. Supports Remote Firmware Updates (FOTA).
- **Tech:** Pure C, TivaWare, UART, ADC, I2C, MQTT, ThingsBoard Cloud.

### 🏃 WeRun & WeRun 2.0 (Mobile Fitness)
Fitness running tracker with advanced analytics and social vibes.
- **WeRun (v1):** Native Android app built with Kotlin, Jetpack Compose, and Firebase for core GPS and run history tracking.
- **WeRun 2.0:** Complete architectural overhaul using **Flutter** to incorporate cross-platform social feeds, achievement goals, and real-time leaderboards.

### 🤖 Autonomous Stair-Climbing Line-Follower Robot
A differential-drive autonomous vehicle capable of:
- Detecting & climbing physical stairs smoothly while maintaining balance mid-step.
- Automatic mode switching and re-locating the navigation line post-climb.
- **Tech:** ESP32 + FreeRTOS task control + PID tuning on 5-array sensors + ultrasonic distance fusion logic.

### 🍔 FoodHere Delivery App
A colorful & modern food delivery platform featuring native Android UI components.
- Integrated VNPay / SePay / MoMo gateways for automated payment processing.
- Real-time driver tracking and interactive routing utilizing the **Mapbox SDK**.
- **Tech:** Kotlin, Jetpack Compose, REST APIs, WebSockets.

### 🧠 Facial Emotion Recognition using Deep Learning
A scientific research (NCKH) project aimed at analyzing human emotions in real-time.
- Utilizes YOLO for precise face detection and Transformer architectures for sequence-based emotion classification.
